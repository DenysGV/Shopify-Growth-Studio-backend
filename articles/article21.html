Створення та підключення Facebook Pixel до Shopify

Facebook Pixel – це аналітичний інструмент, який дозволяє відстежувати дії відвідувачів на вашому сайті та оптимізувати рекламні кампанії. Він допомагає Facebook Ads розуміти, як користувачі взаємодіють з вашим магазином, і показувати їм більш релевантну рекламу.

Створення Facebook Pixel

Перейдіть у Facebook Events Manager

  • Відкрийте Facebook Events Manager.
  • Натисніть Connect Data Sources → Web → Facebook Pixel.
  • Введіть назву для пікселя (наприклад, "Shopify Pixel") та натисніть Create Pixel.

Виберіть платформу Shopify

  • Виберіть Use a Partner Integration.
  • Натисніть Shopify та дотримуйтесь інструкцій.

Підключення Facebook Pixel до Shopify

Увійдіть в Shopify Admin

  • Перейдіть у Online Store → Preferences.
  • Прокрутіть вниз до Facebook Pixel.
  • Вставте Pixel ID, який ви отримали в Facebook Events Manager.
  • Натисніть Save.

Переконайтеся, що Facebook Pixel працює

  • Встановіть розширення Meta Pixel Helper для Chrome.
  • Відкрийте свій сайт і перевірте, чи активний Pixel.

Налаштування подій

Перейдіть у Facebook Events Manager.

  • Виберіть ваш Pixel.
  • Натисніть Open Event Setup Tool.
  • Введіть URL вашого магазину та натисніть Open Website.

Додайте події.

Add to Cart, Purchase та збережіть зміни.

Перевірка роботи Pixel

  1. Відкрийте Facebook Events Manager.
  2. Переконайтеся, що з’являються події (наприклад, перегляд товару, додавання в кошик).
  3. Перейдіть до Shopify і зробіть тестову покупку.
  4. Перевірте, чи зафіксована подія "Purchase" у Facebook Events Manager.

Готово! Тепер Facebook Pixel підключений до Shopify і готовий відстежувати дії користувачів для оптимізації реклами.
